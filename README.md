CampusHub: The Oracle of a Strong Marketplace Architecture

CampusHub is a sophisticated Android application developed in order to reconcile the most recent principles of e-commerce with student-centered campus marketplaces. With the focus on reliability and "Spaceship" inspired techno-design, CampusHub ensures a reliable basis for any student-to-student or campus-to-student trade.

Features and Design Specifications
A Comprehensive Set of Icons and UI Components: Includes the set of icons (more than 60 custom-made) and other UI elements required by the application to ensure its e-commerce performance.

Screens Deep Architecture: Includes not less than 20 screens, namely News/Offer screens and some other dedicated to products and users' profile management.

Clean Code Principle: Comments included in each logical piece of code in the XML and Java source code of the application.

Current Trends Oriented: A technologically advanced design solution based on the current web and mobile interface design trends ("Digital Interior" style).

Consistent Design: Clear and logical naming, grouping and positioning of all app's layers for easy reading and manipulation.

Technical Stability and Consistency
Protection against Crashes: All RecyclerViews, adapters and cart management algorithms designed with regard to null safety rules to prevent any crashes when switching accounts or operating with the lack of information.

Dynamic Cart Management: A dedicated checkout system with a real-time updating and calculating of quantity and prices/subtotals of the cart.

Top Navigation Bar: Custom fragments' management framework implemented for smooth navigation through DrawerLayout and Top Toolbar.
